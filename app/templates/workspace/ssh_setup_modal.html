Setup Instructions

  1. 1

    Copy the SSH public key above

    Use the "Copy" button or click the key to select all and press Ctrl+C (Cmd+C on Mac)

  2. 2

    Go to GitHub SSH Keys Settings

    Open GitHub → Settings → SSH Keys in a new tab

  3. 3

    Add new SSH key

    • Title: {{ workspace.subdomain }} - YouAreCoder

    • Key type: Authentication Key

    • Key: Paste your copied SSH public key

  4. 4

    Click "Add SSH key" on GitHub

    You may be prompted to confirm your GitHub password

  5. 5

    Verify SSH connection and open workspace

    Click the "Verify & Open Workspace" button below to test the connection

Important Notes:

  • This SSH key is unique to your workspace
  • Keep this key in your GitHub account while using this workspace
  • You can add the same key to multiple GitHub accounts if needed
